# Server side
python flask code

##### Note: run npm run build in frontend to see changes when testing server side!!!

## install
``` 
    pip install flask   or conda install flask
    pip install python-dotenv or  conda install -c conda-forge python-dotenv
```

## run
```
    cd server
    flask run
```
